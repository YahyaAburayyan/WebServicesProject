LRMIS

Applications

POST Create Application

Submits a new land registration application. Returns a unique LRMIS-2026-XXXX id.

GET List Applications

Returns a paginated, filterable list of all applications.

GET Get Application by ID

Fetches the full record for a single application.

PATCH Transition Status

Moves an application to a new status. Only legal transitions are allowed.

Tip: use Get by ID first to see which transitions are currently allowed for your application.

POST Put on Hold

Pauses processing of an application. A reason is required.

POST Reject Application

Permanently rejects an application. A reason is mandatory. This is a terminal state.

POST Issue Certificate

Issues a land certificate. The application must be in approved status first.
